风扇模式切换策略设计

本模块实现光伏逆变器的智能风扇控制策略，根据实时温度数据动态调整风扇转速，以达到最优的散热效果与能效平衡。

设计目标：

  • 支持 5 级风扇模式：静音、低速、中速、高速、全速
  • 温度滞回控制，避免频繁切换
  • 支持手动模式覆盖与自动恢复
  • 异常保护：传感器故障时自动进入安全模式
  • 实时性要求：模式切换响应时间 < 100ms

关键接口：通过 PWM 输出控制风扇转速，采样周期 500ms，支持外部温度传感器扩展。
